## 简介

**宇浩·星陈**是宇浩输入法的次世代衍生方案，具有强聚类、强音托、弱分区的特点。它同宇浩二代共用一套拆分表，只是字根大小码不同，故而并行维护。

[宇浩·星陈官网](http://zhuyuhao.com/yustar/)
